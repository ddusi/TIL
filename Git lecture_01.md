# CLI

커맨드 라인 인터페이스

### 터미널 명령어들

- `ls` : 폴더 내부의 파일 & 폴더를 나열

- `pwd` : 현재 폴더 경로 출력

- `mkdir` + [폴더명] : 폴더 생성

- `cd` + [폴더명] : 폴더 변경

- `cd ..` : 상위폴더로 변경

- `git init` :  폴더 관리해줘.

- `ls -a` : 숨김 파일도 보여줘

- `git status` : git의 상태

- `touch` + [파일명.확장자] : 파일 만들기

- `rm` + [파일명] : 파일 삭제
