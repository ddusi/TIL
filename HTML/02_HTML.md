# HTML

### 1. HTML의 기본 구조

```html
<!doctype html>
<html lang="ko">
 	<head>
 ...
 	</head>
     <body>
 ...
 	</body>
</html>

```

 

#### `<Head>`태그

- 웹 브라우저 화면에는 보이지 않지만, 웹 브라우저가 알아두어야 할 정보들 입력 

- 문서에서 사용할 외부 파일들 링크



`<title>`태그 : 브라우저의 제목 표시줄에 표시되는 내용

`<meta>`태그 : 문자 인코딩 방법 및 문서의 키워드와 요약 정보를 지정

​			`<meta charset="utf-8">`
